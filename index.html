진짜 E2E 암호화

경쟁 앱 5개 모두에 없는 기능. 메시지·이미지·화면 캡처 모두 단말 간 암호화로 보호됩니다.

  • 🔑
    기기별 RSA 키 쌍 자동 생성
    최초 로그인 시 Android Keystore · iOS Secure Enclave · Windows DPAPI에 개인키 저장. 앱 외부 추출 불가.
  • ⚡
    AES-256-GCM 콘텐츠 암호화
    매 전송마다 새 세션키 생성(Perfect Forward Secrecy). GCM 인증 태그로 위변조 자동 검증.
  • 🛡️
    TLS 1.3 이중 적용
    E2E 암호화에 더해 전송 구간도 TLS 1.3으로 보호. 네트워크 도청 이중 차단.
  • 🗑️
    수신 확인 즉시 삭제
    이미지·화면 캡처는 Presigned URL(유효 5분) 방식. 수신 완료 후 서버에서 즉시 삭제.

시작하기

4단계로 완료

복잡한 설정 없이 QR 스캔 한 번으로 연결됩니다.

1
앱 설치 & 계정 생성

Google Play / App Store에서 GuardPact를 설치하고 이메일·Google·Apple로 가입.

2
QR 코드 생성

원격관리자 계정에서 구성원 추가 버튼을 누르면 QR 코드 자동 생성 (유효 30분).

3
구성원 기기 연결

구성원 기기에서 QR 스캔 → 권한 허용 → 즉시 연결 완료. 여러 기기 동시 등록 가능.

4
정책 설정 & 관리

앱별 시간 제한, 차단 목록, 취침 모드를 설정하면 구성원 기기에 즉시 적용.
